# Entertainment & Media Analytics Dashboard

## Project Overview
The Entertainment & Media Analytics Dashboard is an interactive Power BI project developed to analyze movies and TV series using IMDb-based datasets. The dashboard provides insights into content ratings, genres, cast information, audience engagement, success classification, smart recommendations, streaming availability, and AI-assisted user interaction.

This project is designed to help users explore entertainment content more effectively through data visualization and interactive filtering.

---

## Objectives
- To analyze movies and TV series using structured entertainment data
- To provide visual insights into ratings, genres, cast, and audience popularity
- To classify content performance as Hit, Average, or Flop
- To build a recommendation-based dashboard for better content discovery
- To enhance user interaction using streaming platform information and AI assistance

---

## Key Features
- Interactive Power BI dashboard
- Analysis of movies and TV series
