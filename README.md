## Project Overview
CineMatch AI is an intelligent entertainment discovery and analytics platform designed to help users explore movies and TV series more effectively through interactive data analysis, recommendation support, and AI-assisted user interaction.

The project combines:
- an interactive **Power BI analytics dashboard**
- an **AI-powered interactive portal**
- entertainment metadata and ratings analysis
- recommendation-oriented content exploration
- streaming platform availability support

This project was developed as an academic mini-project / major project to demonstrate the practical application of **Business Intelligence**, **interactive analytics**, and **AI-assisted entertainment discovery**.

---

## Main Objective
The main objective of this project is to create a unified entertainment exploration platform where users can analyze and discover movies and TV series based on genre, year, rating, cast, runtime, popularity, streaming availability, and recommendation logic.

---

## Project Modules

### 1. Power BI Analytics Dashboard
This module provides visual and analytical insights into movies and TV series using charts, KPIs, slicers, tables, and recommendation-oriented views.

### 2. AI Interactive Portal
This module provides a web-style interactive interface for entertainment discovery, AI-assisted guidance, and recommendation support.

### 3. Data Module
This module includes IMDb datasets and supplementary streaming platform data used for analysis and content mapping.
